# Thinking Tools Collection

A curated collection of mental models, thinking frameworks, and decision-making tools used by the world's best thinkers, investors, and leaders. Each tool includes a description, when to use it, step-by-step instructions, and real-world examples.

## Why Collect Thinking Tools?

Charlie Munger, Warren Buffett's partner, famously advocates for building a "latticework of mental models" — a diverse toolkit of thinking frameworks drawn from multiple disciplines. The person with only a hammer treats every problem as a nail. The person with a full toolkit can match the right tool to each situation.

## Featured Tool: Inversion

> "Invert, always invert." — Carl Gustav Jacob Jacobi

Instead of asking "How do I succeed?", ask "How would I guarantee failure?" Then avoid those things.

**Steps:**
1. Define your desired outcome
2. Ask: "What would guarantee the opposite?"
3. List all the ways you could fail
4. For each failure mode, define its opposite as a rule
5. Follow those rules

**Example — Building a great team:**
- Ways to guarantee a bad team: hire for credentials not values, avoid difficult conversations, tolerate low performers, create unclear ownership
- Inverted rules: hire for culture fit, have candid feedback loops, address performance issues within 30 days, define clear ownership for every project
